[![View SubnetSphere](https://img.shields.io/badge/View%20SubnetSphere-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/hiruranasinghe/SubnetSphere.git)

---

### 🏢 Bonalbo Branch Network — VLAN Design
A Cisco Packet Tracer project focused on designing a branch office network using VLAN segmentation, subnetting, and structured LAN architecture. Demonstrates secure departmental separation and scalable network design.

[![View Project](https://img.shields.io/badge/View%20Project-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/hiruranasinghe/bonalbo-branch-network-vlan-design)

---

### 🏨 Vic Modern Hotel Network (VMH-OSPF-VLAN-Architecture)
A comprehensive enterprise network simulation for a multi-floor hotel using VLAN segmentation, inter-VLAN routing, and OSPF. Features dynamic IP allocation (DHCP), secure SSH access, wireless connectivity, and port security implementation.

[![View Project](https://img.shields.io/badge/View%20Project-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/hiruranasinghe/VMH-OSPF-VLAN-Architecture)
